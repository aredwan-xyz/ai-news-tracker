# AI News Tracker

Automatically fetches and archives daily AI news from top sources using GitHub Actions.

## Sources
- MIT Technology Review
- VentureBeat AI
- AI News (artificialintelligence-news.com)
- The Verge AI

## How it works
A GitHub Actions workflow runs every day at 8am UTC, fetches the latest headlines, and commits a markdown file to the `news/` folder.

## Browse news
All daily snapshots are in the [news/](./news/) folder.
